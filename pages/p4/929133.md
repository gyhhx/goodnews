   奇特指向性、目的性
  </strong>
  <br>
   首先来说一说瘟疫的一个奇特的规律和特性，古往今来历次重大瘟疫，都有其特定的指向人群，病毒的发作都集中在某一特定人群的身上，以下是几个例子
   <br>
    <br>
     <strong>
      1.明末大瘟疫
     </strong>
     <br>
      明朝末年的鼠疫非常清晰明确的指向明朝大军，李自成在与明朝军队近距离作战时军队被感染的很少，清军八旗兵将更是一个都没有，更神奇的是得了瘟疫的明朝部队，只要投降了清军，投降后瘟疫也消失了，当李自成打进京城时，瘟疫已经把明朝军队消灭了十之八九，顺治京城登基后，在京城肆虐一年多的瘟疫突然就此消失，无影无踪。
      <br>
       <br>
        <strong>
         2.西班牙流感
        </strong>
        <br>
         西班牙流感指向性也非常明确，指向青壮年，目的是结束一战。古往今来还没有哪个瘟疫是专门针对青壮年的，西班牙流感可谓独树一帜，但和它的目的性结合起来看，为了结束战争的话，这个指向还是非常有效的。
         <br/>
         <br/>
         <strong>
          3.欧洲黑死病
         </strong>
         <br/>
         黑死病是惨烈到人们甚至不愿意提及和回忆的大瘟疫之一，黑死病带走了欧洲三分之一的人口，使整个欧洲平均寿命降低了10岁，但黑死病也是有指向有选择性的，瘟疫不但针对某些特定人群，也会刻意避开某些人，例如明末瘟疫刻意避开清军，黑死病爆发时少数国家地区也神奇的未被感染。位于重灾区的城市米兰，周围其他城市均被感染了，被黑死病重重包围的米兰竟奇迹般安然无恙；布拉格也神奇地避过了它，它也没有波及到芬兰。不可捉摸的是，当时得以幸免的冰岛百年后才被它侵袭，它对瑞典的侵袭也推迟了两年之久。
         <br/>
         <br/>
         我们现在回头总结、对比数据就会发现，黑死病指向的是当时一部分堕落的只为钱，已经不能称为教会的基督教会，一部分权钱交易的假基督徒，一部分包养情人 生活混乱的不是神职人员的神职人员。

  分析这个问题之前，我们先来看一下美国的总体情况，为什么美国成了全球疫情最严重的国家，难道这次新冠是专门指向欧美国家吗？答案：是，也不是。

  <br/>
  中美建交近50年，而中国经济高速发展的这最近30年，与美国的四位总统脱不开干系。
  <br/>
  <br/>
  <strong>
   第一位：老布什
  </strong>
  <br/>
  老布什曾经代表美国政府驻扎在北京一年多，相当于当时的美国驻华大使，所以他对中共有一种亲近感，六四之后当其他国家都在谴责中共将坦克开上街头，屠杀学生的时候，老布什却偷偷的跟邓小平私下里讲：我是迫于舆论压力才对你们经济制裁的，并向邓承诺会全力帮助中共发展经济，随后又推动中共加入世贸组织（WTO）,但是因为有了六四事件，全球的民主国家及世贸组织成员对中共的要求是必须改善人权，能否进世贸是跟人权状况挂钩的，所以中共一直卡在这里谈判非常艰难，但是转折点不久之后到了，老布什下台了。
  <br/>
  <br/>
  <strong>
   第二位：克林顿
  </strong>
  <br/>
  随后上台的克林顿没有经过太多谈判直接给了中共永久最惠国待遇。等于把中共的人权问题扔到一边了，克林顿的中心思想是允许中共进入世贸的话，至少它会打开国门，至少能向自由和民主靠近一步，而不是像现在这样封国锁门的状态。其他国家买了克林顿的帐，最终使得中共进了世贸，加入世贸是中共经济能够高速发展的一个重要因素。
  <br/>
  <br/>
  <strong>
   第三位：小布什
  </strong>
  <br/>
  小布什在任期间发生911事件，将大部分精力放在反恐上的小布什对中共的态度基本延续了克林顿的策略，为了得到中共在反恐方面的国际支持，小布什甚至把江泽民请到他的私人农场里吃烤肉，并且就从小布什开始，美国华尔街开始接纳中国企业上市，实际上就是允许中共在美国圈钱。
  <br/>
  <br/>
  <strong>
   第四位：奥巴马
  </strong>
  <br/>
  奥巴马上任伊始拿不准该如何对待中共，结果他做了个非常离谱的事情，直接要中共列一个清单，内容是中共你希望美国如何对待你，所以美国后来的对华政策有一半是中共制定的，奥巴马期间美国门户全面对中共开放，签证也由半年一签变为十年一签，中共大量物资在美国倾销，美国诸多财团、500强企业到中国投资、建厂、创造了大量的就业与经济收入。
  <br/>
  <br/>
  30年间的四任美国总统，牺牲了美国的大量利益，换来了中共的高速经济发展，美国之所以花这么大的精力来帮助中共，是基于以前的一个成功经验。
  <br/>
  <br/>
  二战刚结束时，美国的GDP占了全球的60%，而当时饱受战乱凌虐的欧洲是满目疮痍百废待兴的状态，美国当时的策略是对欧洲提供大量的无偿援助，制定并实施了一个马歇尔计划，美国希望欧洲通过经济的繁荣能够变成一个民主自由的地区，能够成为一个繁荣的市场，事实上也最终也实现了这一点。
  <br/>
  <br/>
  所以同样的做法美国也想用在中共身上，希望中国通过开放市场、经济富强后也能够实现民主和自由，但是在中国身上这个策略就出问题了，中共是个集权独裁的体制，不管他有钱没钱都无法改变这个性质，有了钱之后的中共，在西方收买高科技企业，偷窃技术帮它在国内建立封锁互联网的技术，建立全民监控的网络，有了钱之后的中共裹挟着全体中国人一切向钱看，物欲至上、贪婪狭隘的社会反而人权状况更糟糕。
  <br/>
  <br/>
  此时此刻，疫情最严重的纽约州，便是集中了大量曾经被中共收买的高科技企业，支持过中共为其输送大量资金或投资的金融企业，为其输送利益的信用评估机构。

  病毒为什么集中在民主党所在地区？
  <br/>
  <br/>
  代表民主党竞选的最终的两位竞选人：伯尼·桑德斯（下文简称桑德斯）、乔·拜登（简称拜登）。这两位如果分别用一句话来介绍和形容的话，应该是这样的：
  <br/>
  桑德斯：崇尚社会主义，如果当选总统将带领美国走社会主义道路。
  <br/>
  拜登：其基金会曾收受中共15亿美金，因贪腐丑闻一直在被调查。
  <br/>
  <br/>
  不但桑德斯崇尚社会主义，这一次民主党参加竞选的22位竞选人几个每一个都是提倡和倒向社会主义的，整个民主党被中共的意识形态的渗透，和经济利益的收买，已经变成一个倒向社会主义，严重倾向共产主义的一个政党。
  <br/>
  <br/>
  这两位民主党竞选人，不论哪个当选总统，最终结果一定是亲近中共，会带领美国走共产党正在走的路，甚至有可能把美国变成社会主义国家。
  <br/>
  眼下美国和中共意识形态的对立，是川普上任之后的事，尤其是中美贸易战和香港问题之后形成的，但在此之前的30多年时间里，美国的四任总统从经济上、物资上、发展上给了中共大量的、长时间的支持。
  <br/>
  <br/>
  所以跟中共走的近的，支持中共的，意识形态贴近中共的，为其输送利益的是不是就是这次新冠所指向的目标人群呢？我们先把结论先放在这里，看看除了美国之外的其他国家。
  <br/>
  <br/>
  疫情严重程度紧随美国之后的是欧洲的几个国家，大家知道以前的东欧是在前苏联共产主义的控制之下，目前东欧有些国家完全解体抛弃共产主义的前东欧，这些国家的感染人数大多为个位数，很少超过百位。而感染人数最多的几个国家，意大利、西班牙、法国，都是与中共走的很近的国家，我们来具体看一看。
  <br/>
  <br/>
  <strong>
   1.意大利
  </strong>
  <br/>
  意大利是全欧洲第一个加入中共一带一路的国家，也是全球七大工业国（G7）首个加入“一带一路”的，意大利加入时其他国家都劝他不要跟中共太近，但是意大利还是执意要加入，意大利十年间接受中共230亿欧元，意大利的普拉托（Prato）四分之一的人口都是温州人，很多国家把意大利看成是中共渗透欧盟的特洛伊木马。
  <br/>
  <br/>
  <strong>
   2.西班牙
  </strong>
  <br/>
  89年六四之后，第一个正式派人访问中国的就是西班牙，西班牙建了八所中共专门输出意识形态的孔子学院，05年与中共建立全面战略伙伴关系，虽未正式参与一带一路，但在部分项目上有合作。08年金融海啸时，中共购买12%西班牙国债，成为其第二大债权国。
  <br/>
  <br/>
  <strong>
   3.德国
  </strong>
  <br/>
  德国北威州（Nordrhein-Westfalen）是最亲共的州，中国进驻德国的3000家企业中，有1100家三分之一在北威州。华为、中兴欧洲总部都在北威州。2019年北威州从中国进口281亿欧元，占全德国1/4；北威州出口到中国118亿欧元，占德国对华出口1/8。北威州也积极参与“一带一路”，北威州有20个城市与中国结为姐妹城市、200座大学结为姊妹大学。
  <br/>
  <br/>
  4.法国
  <br/>
  法国是第一个与中共建交的西方大国，现任总统马克龙曾说他是“毛泽东主义者”。 前总理拉法兰帮中共建立武汉病毒研究所P4实验室。法国在2019年与中共签署15项经贸合同，价值400亿欧元。100多家法国企业在武汉投资，雪铁龙就在武汉。
  <br/>
  <br/>
  <strong>
   5.伊朗
  </strong>
  <br/>
  中共是伊朗最大贸易伙伴。中共承诺在未来25年，向伊朗的石油天然气等产业投资4000亿美元，去年贸易额多达134亿美元，中国也是伊朗最大输出国，占伊朗产值50～70%。
  <br/>
  <br/>
  伊朗自1979年“伊斯兰革命”后与欧美疏离，转与中共交好，中共成为伊朗后台，暗中扶植伊朗在中东地区牵制美国、以色列等国家。伊朗被美国制裁后，中国仍大量采购伊朗原油，为其提供经济资助。
  <br/>
  <br/>
  中共长年对伊朗提供导弹、战机、潜艇等武器，并秘密帮助伊朗建立化学武器项目。幕后支持伊朗研发核武器，伊朗也是中共“一带一路”的关键枢纽。
 </p>
 <p>
  篇幅有限，其他国家不再一一分析，以上为反面教材，这几个与中共亲近、为其输送利益，为其站队、给它投资送钱的国家，实际的现状就是疫情爆发最严重，病毒肆虐最厉害的地区。
  <br/>
  <br/>
  接下来是正面教材，我们看看远离中共的国家是什么状态。
  <br/>
  <br/>
  <strong>
   1.台湾
  </strong>
  <br/>
  台湾多年来长期饱受中共的各种恐吓，包括言论、经济与武力等各方面，所以台湾可能是最不相信中共，对中共最避而远之的，当李文亮等医生发出消息时，尽管中共裹挟着世卫组织一起宣布不会人传人，但台湾根本不信，第一时间断绝了与大陆的一切接触，隔断了传播途径。时至今日当其他国家感染人数动以万计之时，台湾只感染了300多人，死亡仅5人。

  <br/>
  <strong>
   2.蒙古
