<div><p>
 【大纪元2020年04月03日讯】
 <span class="s1">
  （大纪元记者骆亚采访报导）近期大陆多地出现抢粮潮，中共官方声
 </span>
 <span class="s1">
  称外国限制出口不会对国内造成影响。
 </span>
 <span class="s1">
  不过大陆“水稻之父”袁隆平则表示，如果外国不售粮的话，中国粮食不够吃，就要饿肚子。多地民众也表示中共诚信缺失，官方越辟谣越说明是真的，所以都会囤一些以防
  <ok href="https://www.epochtimes.com/gb/tag/%E7%B2%AE%E8%8D%92.html">
   粮荒
  </ok>
  。
 </span>
</p>
<h4 class="p3">
 <span class="s1">
  中共官方：按需购买不必超期囤购
 </span>
</h4>
<p class="p3">
 <span class="s1">
  据陆媒报导，大陆全国多地出现抢购粮食的现象，多家超市的粮食宣告售罄。近两日北京店内的米、面销售量也比平时提高
 </span>
 <span class="s4">
  50%
 </span>
 <span class="s1">
  。
 </span>
</p>
<p class="p3">
 <span class="s1">
  报导说抢购潮源于多国纷纷禁止粮食出口的新闻，微信疯传之后，开始出现米油即将涨价的说法。
 </span>
</p>
<p class="p5">
 <span class="s1">
  安徽、重庆、湖北及国家粮食储备局等连日来纷纷“辟谣”称，粮油市场供应充足、产能稳定，希望居民按需购买，不必超期囤购，并强调，“个别国家限制出口，对国内的供应基本没有影响。”
 </span>
</p>
<h4 class="p5">
 <span class="s1">
  专家：外国禁出口
 </span>
 <span class="s1">
  中国人要饿肚子
 </span>
</h4>
<p class="p5">
 近日网络热传一段杂交水稻育种专家袁隆平去年7月接受官媒专访的视频，他的观点正好与中共官方的言论相左。
</p>
<p class="p5">
 <span class="s1">
  他坦言一旦人家不卖给中国粮食，中国就会出大问题，“中国的粮食是不够吃的。中国差不多
 </span>
 <span class="s5">
  14
 </span>
 <span class="s1">
  亿人口，粮食是不够吃的，要进口一部分。我举个例子，大豆不够吃，我们中国人吃的豆腐，每年进口大豆七八千万吨，现在国家还有钱买粮食，如果人家一卡不卖了，那就麻烦了，要饿肚子了，这是大问题。”
 </span>
</p>
<h4 class="p7">
 <span class="s1">
  武汉体制内人：政府辟谣了也要囤
 </span>
</h4>
<p class="p3">
 <span class="s1">
  家住湖北武汉的大学生吴同学向大纪元表示，“我爸妈是体制内的，感觉平时挺信任党的，但是这次我爸说肯定要囤粮，政府辟谣了也要囤。我妈也同意我爸的观点，囤一些有备无患，所以我们家现在囤了一些米，还买了些罐头。就是
 </span>
 <span class="s4">
  10
 </span>
 <span class="s1">
  斤一袋的米先买了10袋，之后看情况接着买。油的话家里还有几筒，暂时没买。”
 </span>
</p>
<p class="p3">
 <span class="s1">
  他还介绍自己远在美国的亲戚也囤粮、买罐头，感觉华人好多都喜欢囤东西。
 </span>
</p>
<p class="p3">
 <span class="s1">
  他也认为各地政府在
  <ok href="https://www.epochtimes.com/gb/tag/%E7%B2%AE%E8%8D%92.html">
   粮荒
  </ok>
  问题上撒谎，“从中国依赖粮食进口的情况看，周边产粮国禁止粮食出口对中国肯定是有影响的，再加上现在国内一些粮食产区产粮受
  <ok href="https://www.epochtimes.com/gb/tag/%E7%96%AB%E6%83%85.html">
   疫情
  </ok>
  影响被耽误，会造成减产，粮食供应确实很可能出现紧张，民众出于担忧抢购粮食也是有道理的。当局不顾及上述因素，一味干巴巴地强调‘库存足够，勿信谣言’是不合适的，再加上中共这些年信用破产，中共的‘辟谣’也是难以服众的。”
 </span>
</p>
<h4 class="p3">
 <span class="s1">
  河南人：当地没有抢购我也会囤一点
 </span>
</h4>
<p class="p3">
 <span class="s1">
  河南的杰克向大纪元记者介绍，当地还没有抢购粮食的现象，“我在网上听说是武汉那边有这种现象发生。我也会囤一点，不然，真有粮荒的那么一天，不知道怎么办了。而且全球的
  <ok href="https://www.epochtimes.com/gb/tag/%E7%96%AB%E6%83%85.html">
   疫情
  </ok>
  这么严重，中国未来很有可能会出现粮荒的。”
 </span>
</p>
<h4 class="p3">
 <span class="s1">
  佳木斯人：我储备的粮食够吃一年的
 </span>
</h4>
<p class="p3">
 <span class="s1">
  佳木斯地区的一位民众李先生告诉大纪元记者，“这里虽然也有多购粮的，但是还没有抢购的风潮。只是疫情刚解封，又出现了多起感染者病例，又开始封小区了。”
 </span>
</p>
<p class="p3">
 <span class="s1">
  他介绍因为佳木斯是产粮基地，所以粮食在这里还不是那么紧张。不过他给自己储备的粮食够吃一年的。
 </span>
</p>
<p class="p3">
 <span class="s1">
  他认为中国会有很大的粮荒，“因为全世界天灾频频，中国大陆执政者不顺天意，致使天灾人祸不断，所以，今年一定不会风调雨顺的。”
 </span>
</p>
<p class="p3">
 <span class="s1">
  他强调，“现在党媒不断地辟谣，说粮食充足，依我看，他们越是这样宣传，就说明心虚，是用这种方式求得安定，以往的规律就是，党媒说往东，你就往西，这就是真理。”#
 </span>
</p>
