 市民：说这公交车站，他跨公交车道，跨实线。
 <br/>
 交警：这个时间段，他是有特权的。
 <br/>
 市民：没有特权。中国，他们是公民，不，他们是公仆。
 <br/>
 交警：别嚷嚷，我跟你没仇，你别跟我嚷嚷。
 <br/>
 市民：有什么话说的，罚他呀。
 <br/>
 交警：别嚷嚷，你不用嚷嚷。
 <br/>
 市民：我冲他说的，罚他呀。你是公仆，记住了，再怎么着，你是公仆，你是人民选出来的，你也不能走自行车道，你有特权是吗？
 <br/>
 人大代表：我没说。
 <br/>
 市民：你跟我说有特权是吗？光天化日之下，长安街上有特权，我不信，我就不信啦。有特权是吗？
 <br/>
 交警：冷静点，冷静点。
 <br/>
 市民：没有车站，他停在这儿，我看不见车站。
 <br/>
 交警：你听说我，车证，车号我给你记下来。
 <br/>
 市民：不，告诉您几回了，罚他。您查证件，他验车了吗？他有驾照吗？他喝没喝酒？
 <br/>
 交警：他这个东西是正常走，我们之后会查他。
 <br/>
 市民：别，现在啊，等他什么之后啊？
 <br/>
 交警：我们向组委会反映之后。
 <br/>
 市民：没有组委会，没有组委会，今天两会第一天，我就不明白了，长安街跨公交车道，走自行车道实线，告诉我有特权，他有证儿吗？
 <br/>
 交警：这不写着呢。
 <br/>
 市民：他有驾照吗？您能查一下吗？他喝没喝酒啊？出示一下啊？
 <br/>
 交警（对人大代表）您把证件给他看一下，有没有驾照。
 <br/>
 市民：不不，您看，我不看。
 <br/>
 交警：您要求要看啊。
 <br/>
 市民：您查一下，他违法了。
 <br/>
 交警：违法了，组委会对他进行处理。
 <br/>
 市民：您是交警，没有组委会，没有组委会。
 <br/>
 交警：现在，两会，就让他先开会，我们发到组委会。
 <br/>
 市民：开什么会呀，他从后边，两会，他是红旗车，我没有车牌，他丢的是红旗的脸，这么多人看着呢，两会期间，长安街，从自行车道走，我骑自行车，你从我身边蹭过去啊？吓着我，是不是？
 <br/>
 交警：蹭着您了吗？
 <br/>
 市民：没有，但是我怎么知道自行车会得让辆汽车啊？我怎么知道自行车道会得有辆汽车出现啊？
 <br/>
 交警：您是今天上午参加什么会啊？（问人大代表）
 <br/>
 市民：您干吗呀您？你有这么能吗？而且他在右拐线，他左拐，直行是吗？你说什么？你说什么？
 <br/>
 交警：您开会去，是吗？
