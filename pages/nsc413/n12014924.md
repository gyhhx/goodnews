  <ok href="https://www.epochtimes.com/gb/tag/%E7%99%BE%E5%AE%B6%E7%81%B0.html">

  ”事件曝光后，更是激起民愤。

</figure><br/>
<p class="p10">
 <span class="s9">
  所谓百家灰，是指
 </span>
 <span class="s4">
  武汉家属们领取的亲人骨灰不匹配。
 </span>
 中国异议人士李方向大纪元表示，他收到一个朋友发的截图，说有人曾经打电话问过武昌那边火葬场的一个工人，好像工人的口气很大的，回复就是说，他觉得这很正常。
</p>
<p class="p12">
 李方解释说，“那么多人，一天要烧一百多人，十几个火化炉都不停歇地烧，工作环境很热，温度很高，火化完灰是烫的，刚刚刮出来不可能去装，又没地方堆，灰又很多，所以他们觉得混在一起是很正常的。等灰冷了以后，忙完了，差不多交接班的时候，就再给分一下骨灰，把姓名贴上去。”
</p>
<figure class="wp-caption aligncenter" id="attachment_12015012" style="width: 266px">
 <ok href="https://i.epochtimes.com/assets/uploads/2020/04/guhui.jpeg">
  <img alt="" class="wp-image-12015012" src="https://i.epochtimes.com/assets/uploads/2020/04/guhui-300x538.jpeg"/>
 </ok>
 <br/><figcaption class="wp-caption-text">
  火葬场工人关于“百家灰”的解释。（受访者提供）
 </figcaption><br/>
</figure><br/>
<p class="p12">
 “因为尸体太多了，用车装，有的一车装一、二十个，尸袋上有姓名牌，有的姓名牌路上就丢了，根本不知道烧的是谁？”
</p>
<p class="p12">
 李方指出，其实这种事情不单是因为在武汉死人多，就是在以前正常火化的时候，由于中国是一个缺少信仰的国家，国民普遍来说职业道德很差，如果家属不全程监督，有的根本不确定出来的骨灰是不是自己亲人的。
</p>
<h4 class="p14">
 <span class="s10">
  中共出台“六不得” 索赔
  <ok href="https://www.epochtimes.com/gb/tag/%E5%BE%8B%E5%B8%88%E9%A1%BE%E9%97%AE%E5%9B%A2.html">
   律师顾问团
  </ok>
  受阻
 </span>
</h4>
<p class="p16">
 <span class="s11">
  今年
 </span>
 <span class="s12">
  3
 </span>
 <span class="s10">
  月
 </span>
 <span class="s12">
  6
 </span>
 <span class="s10">
  日，由来自九个省市及旅外访学律师发起的
 </span>
 <span class="s12">
  “
 </span>
 <span class="s10">
  新冠肺炎（中共肺炎）索赔律师顾问团
 </span>
 <span class="s12">
  ”
 </span>
 <span class="s10">
  （以下简称
 </span>
 <span class="s12">
  “
 </span>
 <span class="s10">
  律师顾问团
 </span>
 <span class="s12">
  ”
 </span>
 <span class="s10">
  ）正式成立，为因感染
 </span>
 <span class="s11">
  肺炎
 </span>
 <span class="s10">
  病逝者及医护人员的家属的索赔行动提供法律支持。
 </span>
</p>
<p class="p17">
 作为参与者的李方表示，早在2月底的时候，他们看到中国的老百姓在疫情当中受到了很大的损失和伤害。中共为了武汉市、湖北省两级的“两会”，延迟了2周～3周时间去压制疫情，以致后来疫情大爆发。
</p>
<p class="p17">
 “所以我们就觉得这个责任是在政府，政府造成了疫情大爆发，死亡和后遗症，各种各样的个人、家庭和企业的损失，都是老百姓来承受。我们觉得政府应该负它的责任。由于政府的失误造成老百姓的损失，在任何国家，老百姓都是可以追究国家赔偿的。”
</p>
<figure class="wp-caption aligncenter" id="attachment_12015016" style="width: 394px">
 <ok href="https://i.epochtimes.com/assets/uploads/2020/04/398a5d07dcd6108a35d139dd4cc66c78.jpeg">
  <img alt="" class="wp-image-12015016 size-medium_vertical" src="https://i.epochtimes.com/assets/uploads/2020/04/398a5d07dcd6108a35d139dd4cc66c78-394x400.jpeg"/>
 </ok>
 <br/><figcaption class="wp-caption-text">
  中共司法系统口头要求中国律师“三严禁，六不得”。（受访者提供）
 </figcaption><br/>
</figure><br/>
<p class="p17">
 据他介绍，在三月中旬，中共司法部门专门出了一个“六不得”，不准律师介入到集体诉讼、集体索赔这种案件当中来，因为这个禁令出现以后，律师顾问团在国内地一、二十个律师的行动就受阻了。
</p>
<p class="p17">
 “他们（中共）感觉到了有索赔这么一个动向，所以很快就出台这么一个政策。”李方说。
</p>
<h4 class="p17">
 顾问团受理多起案件 艰难维权
</h4>
<p class="p19">
 <span class="s4">
  中国资深公益人士、律师顾问团的发起人杨占青在接受大纪元采访时表示，目前顾问团收到7、8个受害人的请求，其中三人是因为家属病逝。有的感染者是前期去医院没有床位，等到肺部感染特别严重了终于住上院，但入院一两个小时就死亡了，没有机会确诊。
 </span>
</p>
<p class="p19">
 他说，另外一个感染者是在隔离点死亡的。患者在隔离酒店没有人管，家属去送药，只能送到前台。后来患者病情严重，拖延致死，之前核酸检测阳性。
</p>
<p class="p19">
 <span class="s4">
  有一个人是在感染康复之后，单位不予接收，所以想要维权；还有一个人是第一次检测阳性，但医院说他是病毒性肺炎，不承认是新冠肺炎（中共肺炎）
 </span>
 <span class="s6">
  ，拒绝出示病例档案，他只有出院证明和肺部x光片子，寻求知情同意权。
 </span>
</p>
<p class="p21">
 <span class="s10">
  针对目前国内
 </span>
 <span class="s14">
  压制律师的情况，杨占青表示，
 </span>
 <span class="s10">
  虽然有困难但是还可以做，“因为律师顾问团只是起一个顾问的作用，比如案件需要一些法律意见，但是真正的要起诉，可以另外在国内找律师，跟顾问团是没有任何关系的。”
 </span>
</p>
<p class="p9">
 <span class="s4">
  “所以也不算集体索赔，相当于只是个案，这样没有集体诉讼给官方施加的压力大。但是个案一旦立案，法院不管他是拖着还是审理，都是一个标志性的案件。外界关注这个案子，给政府压力，现在只能通过这种方式去做。”他说。
 </span>
</p>
<p class="p9">
 <span class="s4">
  杨占青认为，在国内维权很艰难，当地对这些维权的人软硬兼施，
 </span>
 <span class="s1">
  一方面是威胁，另一方面又表面给一些好处，比如如果赶快下葬，补贴多少钱，派车免费接送，想通过恩惠把他的案子化解掉。
 </span>
</p>
<p class="p9">
 <span class="s4">
  “由于大环境整体的控制，信息渠道有限，（受害人）需要的话（律师团）提供帮助，他自己不愿意维权也没办法，只能说尽量能做多少做多少。”他说。
 </span>
</p>
